Usage: cq run [OPTIONS]

Execute a multi-step CQ run plan.

╭─ Parameters ─────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ --include --empty-include           Include files matching pattern (glob or ~regex, repeatable) [env var:            │
│                                     CQ_RUN_INCLUDE] [default: []]                                                    │
│ --exclude --empty-exclude           Exclude files matching pattern (glob or ~regex, repeatable) [env var:            │
│                                     CQ_RUN_EXCLUDE] [default: []]                                                    │
│ --impact --empty-impact             Filter by impact bucket (comma-separated: low,med,high) [choices: low, med,      │
│                                     high] [env var: CQ_RUN_IMPACT] [default: []]                                     │
│ --confidence --empty-confidence     Filter by confidence bucket (comma-separated: low,med,high) [choices: low, med,  │
│                                     high] [env var: CQ_RUN_CONFIDENCE] [default: []]                                 │
│ --severity --empty-severity         Filter by severity (comma-separated: error,warning,info) [choices: info,         │
│                                     warning, error] [env var: CQ_RUN_SEVERITY] [default: []]                         │
│ --limit                             Maximum number of findings [env var: CQ_RUN_LIMIT]                               │
│ --stop-on-error --no-stop-on-error  Stop execution on the first step error [env var: CQ_RUN_STOP_ON_ERROR] [default: │
│                                     False]                                                                           │
╰──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
╭─ Global Options ─────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ --root                                    Repository root [env var: CQ_RUN_ROOT]                                     │
│ --verbose -v                              Verbosity level [env var: CQ_RUN_VERBOSE] [default: 0]                     │
│ --format                                  Output format [choices: md, json, both, summary, mermaid, mermaid-class,   │
│                                           dot] [env var: CQ_RUN_FORMAT] [default: md]                                │
│ --artifact-dir                            Artifact directory [env var: CQ_RUN_ARTIFACT_DIR]                          │
│ --no-save-artifact --no-no-save-artifact  Don't save artifact [env var: CQ_RUN_NO_SAVE_ARTIFACT] [default: False]    │
│ --config                                  Config file path [env var: CQ_RUN_CONFIG]                                  │
│ --no-config --no-no-config                Skip config file loading [env var: CQ_RUN_NO_CONFIG] [default: False]      │
╰──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
╭─ Run Input ──────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ --plan                 Path to a run plan TOML file [env var: CQ_RUN_PLAN]                                           │
│ --step --empty-step    Repeatable JSON step object (e.g., '{"type":"q","query":"..."}') [env var: CQ_RUN_STEP]       │
│                        [default: []]                                                                                 │
│ --steps --empty-steps  JSON array of steps (e.g., '[{"type":"q",...},{"type":"calls",...}]') [env var: CQ_RUN_STEPS] │
│                        [default: []]                                                                                 │
╰──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
